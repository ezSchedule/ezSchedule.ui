## Instalação
**Utilize a linha de comando, copie e cole os seguintes comandos:**:
```
cd front-ezchedule

npm i -y 

npm i -y react-router-dom

npm i -y axios
```

## Após isso já pode rodar :)

**Vai la gafanhoto roda esse front lindão:**
```
npm start
```
